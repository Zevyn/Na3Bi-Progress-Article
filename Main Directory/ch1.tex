Hikami-Larkin-Nagaoka (HLN) formula:
\begin{equation}
	\sigma(B) - sigma(0) = \frac{e^{2}}{2\pi h} (ln(\frac{B_{\phi}}{B})-\Psi(\frac{1}{2} + \frac{B_{phi}}{B}))
\end{equation}

the only fit parameter is the phase coherence field $B_{\phi}$. The phase coherence length $L_{\phi}=\sqrt{\frac{h}{8\pi\eB_{\phi}}}$ as a function of the carrier density.
